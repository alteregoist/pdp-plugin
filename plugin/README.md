# Product Detail Page Generator

상품 이미지와 정보를 입력하면 모듈 기반 상세페이지를 자동 생성하는 Cowork 플러그인.

## 기능

- **상세페이지 자동 생성**: 상품 정보를 입력하면 A~F 모듈을 조합하여 상세페이지 HTML을 생성
- **AI 카피라이팅**: 카테고리별 톤에 맞는 상품 카피 자동 작성
- **AI 이미지 생성**: fal.ai를 활용하여 부족한 상품 이미지를 자동 생성/편집
- **멀티 플랫폼 지원**: 롯데마트몰, 쿠팡, 네이버 스마트스토어, 11번가 등 플랫폼별 규격 지원
- **모듈 편집**: 생성된 상세페이지의 모듈 단위 편집, 추가, 삭제, 순서 변경
- **이미지 출력**: Playwright로 HTML → PNG/JPEG 캡처

## 사용법

### 상세페이지 생성
```
/상세페이지 [상품명]
```
대화형으로 상품 정보를 입력하면 상세페이지가 생성됩니다.

### 상세페이지 편집
```
/수정하기 [HTML 파일 경로]
```
기존 상세페이지의 모듈을 편집, 추가, 삭제할 수 있습니다.

### 미리보기
```
/미리보기 [HTML 파일 경로]
```
생성된 상세페이지를 Playwright로 렌더링하여 스크린샷으로 확인합니다. 전체 페이지 또는 모듈별 캡처를 선택할 수 있습니다.

## 모듈 유형

| 모듈 | 이름 | 설명 |
|------|------|------|
| A | 상단배너 | 메인 비주얼 + 상품명 + 핵심 카피 |
| B | 한눈에 보기 | 핵심 특장점 아이콘 카드 |
| C-1 | 상품 컷샷 | 이미지+텍스트 교대 배치 |
| C-2 | 상품 정보 | 텍스트 중심 스펙/성분 정보 |
| D | 인증 정보 | 인증마크 및 설명 |
| E | 인포그래픽 | 수치/비교 데이터 시각화 |
| F | 표시사항 | 법적 고지, 주의사항 |
| IMG | 상품 이미지 | 풀너비 이미지 단독 모듈 |

## 설정

### 환경 변수

| 변수 | 설명 | 필수 |
|------|------|------|
| `FAL_KEY` | fal.ai API 키 | 이미지 생성 시 필수 |

fal.ai API 키는 [fal.ai 대시보드](https://fal.ai/dashboard/keys)에서 발급받을 수 있습니다.

### 시스템 요구사항

- Node.js 18+ (fal.ai MCP 서버, Playwright, 이미지 처리 등 모든 런타임)

필요한 npm 패키지는 첫 실행 시 자동으로 설치됩니다:
- `playwright` — HTML → 이미지 캡처
- `sharp` — 이미지 리사이즈/포맷 변환

## 출력 구조

```
output/
├── {상품명}_detail_page.html     # HTML 파일
├── {상품명}_full.png             # 전체 페이지 이미지
├── images/                       # 사용된 이미지
└── modules/                      # 모듈별 이미지
```

## 기술 스택

- HTML + TailwindCSS (CDN)
- Node.js Playwright (HTML → 이미지 캡처)
- Node.js Sharp (이미지 리사이즈/포맷 변환)
- fal.ai MCP (AI 이미지 생성/편집)
