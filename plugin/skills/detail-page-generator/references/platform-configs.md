# 플랫폼별 상세페이지 설정

플랫폼마다 이미지 사이즈, 모듈 구성, 레이아웃 규격이 다르다.
사용자가 지정한 플랫폼에 맞춰 설정을 적용한다.

---

## 롯데마트몰 (기본)

**이미지 규격**:
- 최대 너비: 860px
- 배너 이미지: 860 x 480px (16:9 근사)
- 상품 컷샷: 860 x 860px (1:1) 또는 860 x 645px (4:3)
- 인증마크 아이콘: 120 x 120px
- 인포그래픽: 860 x auto

**기본 모듈 구성**:
```
A (상단배너) → B (한눈에 보기) → C-1 (컷샷) → C-2 (상품정보) → D (인증) → E (인포그래픽) → F (표시사항)
```

**필수 모듈**: A, C-2, F
**선택 모듈**: B, C-1, D, E, IMG

**특이사항**:
- F 모듈(표시사항)은 반드시 최하단 배치
- 식품 카테고리: D 모듈(인증 정보) 필수
- 전체 이미지 용량 제한 확인 필요

## 쿠팡

**이미지 규격**:
- 최대 너비: 780px
- 배너 이미지: 780 x 440px
- 상품 컷샷: 780 x 780px (1:1)
- 전체 높이 제한 없음

**기본 모듈 구성**:
```
A (상단배너) → B (한눈에 보기) → C-1 (컷샷) → E (인포그래픽) → C-2 (상품정보) → F (표시사항)
```

**필수 모듈**: A, F
**선택 모듈**: B, C-1, C-2, D, E, IMG

**특이사항**:
- 이미지 중심 레이아웃 선호
- 텍스트보다 이미지 비중 높게
- 모바일 최적화 중요 (대부분 모바일 트래픽)

## 커스텀 플랫폼 추가

새 플랫폼을 추가할 때 다음 정보를 정의한다:

```yaml
platform_name: "플랫폼명"
max_width: 860        # px
banner_size: "860x480" # WxH
cutshot_size: "860x860"
icon_size: "120x120"
default_modules: ["A", "B", "C-1", "C-2", "D", "E", "F"]
required_modules: ["A", "F"]
notes: "특이사항"
```

## 모듈 순서 커스터마이징

사용자가 모듈 순서를 변경하려면 `data-module-order` 속성값을 재할당한다.
단, 다음 규칙은 유지한다:
- A(상단배너)는 항상 첫 번째
- F(표시사항)는 항상 마지막
- 나머지 모듈은 자유롭게 순서 변경 가능
