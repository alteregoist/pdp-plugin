# 롯데마트몰 모듈 정의

> 공통 템플릿은 `common.md`를 참조한다. 이 파일에 정의된 항목이 common.md보다 우선 적용된다.

## 플랫폼 기본값

- **최대 너비**: 860px → `max-w-[860px]`
- **기본 모듈 순서**: A → B → C-1 → C-2 → D → E → F
- **필수 모듈**: A, C-2, F
- **특이사항**: F 모듈은 반드시 최하단. 식품 카테고리는 D 모듈 필수.

---

## Module A: 상단배너 (롯데마트몰 전용)

롯데마트몰은 신뢰감·정직함을 강조하는 밝고 깔끔한 스타일을 사용한다.

**오버라이드 포인트**:
- 배너 이미지 규격: 860 × 480px (16:9 근사)
- 텍스트 오버레이 대신 이미지 하단에 텍스트 분리 배치 선호
- 롯데마트 브랜드 컬러(레드 계열) 포인트 사용 가능

```html
<section data-module-type="A" data-module-order="1" class="relative w-full overflow-hidden">
  <img src="./images/banner_main.png" alt="{상품명} 메인 배너" class="w-full h-auto object-cover" />
  <div class="max-w-[860px] mx-auto px-6 py-6 bg-white">
    <p class="text-xs text-gray-400 uppercase tracking-widest mb-1">{카테고리}</p>
    <h1 class="text-2xl font-bold text-gray-900 mb-2 leading-snug">{상품명}</h1>
    <p class="text-gray-600 text-base mb-4">{핵심 한줄 카피}</p>
    <div class="flex items-baseline gap-3">
      <span class="text-red-600 text-2xl font-bold">{판매가}</span>
      <span class="text-gray-400 text-sm line-through">{정가}</span>
      <span class="bg-red-100 text-red-600 text-xs font-semibold px-2 py-0.5 rounded">{할인율}%</span>
    </div>
  </div>
</section>
```

## Module B: 한눈에 보기 (롯데마트몰 전용)

**오버라이드 포인트**:
- 배경: 흰색 (`bg-white`) — 롯데마트몰은 깔끔한 흰 배경 선호
- 아이콘 컨테이너: 사각형 라운드 스타일

```html
<section data-module-type="B" data-module-order="2" class="py-10 px-6 bg-white border-b border-gray-100">
  <div class="max-w-[860px] mx-auto">
    <h2 class="text-lg font-bold text-gray-900 mb-6 text-center">이 상품의 특장점</h2>
    <div class="grid grid-cols-2 md:grid-cols-{N} gap-4">
      <div class="flex items-start gap-4 p-4 rounded-xl bg-gray-50">
        <div class="w-12 h-12 rounded-lg bg-red-50 flex items-center justify-center shrink-0">
          <img src="./images/icon_{N}.png" alt="{특장점}" class="w-7 h-7" />
        </div>
        <div>
          <h3 class="font-semibold text-gray-900 text-sm mb-1">{특장점 키워드}</h3>
          <p class="text-gray-500 text-xs leading-relaxed">{짧은 설명}</p>
        </div>
      </div>
    </div>
  </div>
</section>
```

## Module C-1: 상품 컷샷 (롯데마트몰 전용)

**오버라이드 포인트**:
- 컷샷 이미지 규격: 860 × 860px (1:1) 또는 860 × 645px (4:3)
- 텍스트와 이미지를 상하 배치(세로형) 선호 — 모바일 가독성 우선

```html
<section data-module-type="C-1" data-module-order="3" class="py-10 px-6 bg-white">
  <div class="max-w-[860px] mx-auto space-y-12">
    <div class="flex flex-col gap-5">
      <img src="./images/cutshot_01.png" alt="{설명}" class="w-full rounded-lg" />
      <div class="px-2">
        <span class="text-xs font-semibold text-red-500 uppercase tracking-wider">{키워드 태그}</span>
        <h3 class="text-lg font-bold text-gray-900 mt-1 mb-2">{헤드라인}</h3>
        <p class="text-gray-600 text-sm leading-relaxed">{본문 설명}</p>
      </div>
    </div>
  </div>
</section>
```

## 플랫폼 미정의 모듈

C-2, D, E, F, IMG 모듈은 `common.md` 템플릿을 사용하되 `{MAX_WIDTH}`를 `860`으로 적용한다.
