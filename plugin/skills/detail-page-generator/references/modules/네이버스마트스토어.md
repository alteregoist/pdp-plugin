# 네이버 스마트스토어 모듈 정의

> 공통 템플릿은 `common.md`를 참조한다. 이 파일에 정의된 항목이 common.md보다 우선 적용된다.

## 플랫폼 기본값

- **최대 너비**: 860px → `max-w-[860px]`
- **기본 모듈 순서**: A → B → C-1 → D → E → C-2 → F
- **필수 모듈**: A, F
- **특이사항**: SEO 친화적 텍스트 포함 권장. 이미지 alt 텍스트 중요. GIF 사용 가능.

---

## Module A: 상단배너 (네이버 스마트스토어 전용)

**오버라이드 포인트**:
- 배너 이미지 규격: 860 × 480px
- 네이버 쇼핑 유저는 정보 탐색 성향 — 상품 정보를 바로 노출
- SEO를 위한 텍스트를 HTML에 직접 포함 (이미지 안에만 넣지 않음)
- 네이버 그린(`#03c75a`) 포인트 컬러 활용 가능

```html
<section data-module-type="A" data-module-order="1" class="w-full">
  <div class="relative">
    <img src="./images/banner_main.png" alt="{상품명} - {핵심 한줄 카피}" class="w-full h-auto object-cover" />
  </div>
  <div class="max-w-[860px] mx-auto px-6 py-5 border-b border-gray-100">
    <nav class="text-xs text-gray-400 mb-3">{카테고리} &gt; {서브카테고리}</nav>
    <h1 class="text-xl font-bold text-gray-900 mb-1 leading-snug">{상품명}</h1>
    <p class="text-gray-500 text-sm mb-3">{상품 설명 한줄 — SEO 키워드 포함}</p>
    <div class="flex items-baseline gap-2">
      <span class="text-green-600 text-2xl font-bold">{판매가}</span>
      <span class="text-gray-400 text-sm line-through">{정가}</span>
    </div>
  </div>
</section>
```

## Module B: 한눈에 보기 (네이버 스마트스토어 전용)

**오버라이드 포인트**:
- 정보량이 많은 네이버 유저에 맞게 설명 텍스트 상대적으로 풍부하게
- SEO 키워드를 자연스럽게 본문에 포함

```html
<section data-module-type="B" data-module-order="2" class="py-10 px-6 bg-gray-50">
  <div class="max-w-[860px] mx-auto">
    <h2 class="text-lg font-bold text-gray-900 mb-1">상품 특징</h2>
    <p class="text-gray-500 text-sm mb-7">{SEO 키워드 포함 서브 문구}</p>
    <div class="grid grid-cols-1 md:grid-cols-{N} gap-4">
      <div class="flex gap-4 items-start bg-white p-5 rounded-xl shadow-sm">
        <div class="w-11 h-11 rounded-full bg-green-50 flex items-center justify-center shrink-0">
          <img src="./images/icon_{N}.png" alt="{특장점}" class="w-6 h-6" />
        </div>
        <div>
          <h3 class="font-semibold text-gray-900 text-sm mb-1">{특장점 키워드}</h3>
          <p class="text-gray-500 text-xs leading-relaxed">{설명 — SEO 키워드 자연스럽게 포함}</p>
        </div>
      </div>
    </div>
  </div>
</section>
```

## Module C-1: 상품 컷샷 (네이버 스마트스토어 전용)

**오버라이드 포인트**:
- 좌우 교대 배치 유지 (common.md와 동일하지만 최대 너비 860px 적용)
- alt 텍스트에 검색 키워드 포함

```html
<section data-module-type="C-1" data-module-order="3" class="py-12 px-6 bg-white">
  <div class="max-w-[860px] mx-auto space-y-14">
    <div class="flex flex-col md:flex-row items-center gap-8">
      <div class="w-full md:w-1/2">
        <img src="./images/cutshot_01.png" alt="{상품명} {키워드} 상세 이미지" class="w-full rounded-xl shadow-sm" />
      </div>
      <div class="w-full md:w-1/2">
        <span class="text-xs font-bold text-green-600 uppercase tracking-wider">{키워드 태그}</span>
        <h3 class="text-lg font-bold text-gray-900 mt-2 mb-3">{헤드라인}</h3>
        <p class="text-gray-600 text-sm leading-relaxed">{본문 설명 — SEO 키워드 포함}</p>
      </div>
    </div>
  </div>
</section>
```
