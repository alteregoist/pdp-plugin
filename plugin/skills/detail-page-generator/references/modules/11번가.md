# 11번가 모듈 정의

> 공통 템플릿은 `common.md`를 참조한다. 이 파일에 정의된 항목이 common.md보다 우선 적용된다.

## 플랫폼 기본값

- **최대 너비**: 830px → `max-w-[830px]`
- **기본 모듈 순서**: A → C-1 → B → E → C-2 → D → F
- **필수 모듈**: A, F
- **특이사항**: 컷샷(C-1)이 B보다 앞에 배치. 이미지→정보 순서로 시각적 임팩트 먼저.

---

## Module A: 상단배너 (11번가 전용)

**오버라이드 포인트**:
- 배너 이미지 규격: 830 × 470px
- 11번가는 프로모션·할인 강조 스타일 — 가격 혜택 전면 배치
- 11번가 레드(`#ef2929`) 포인트 컬러 활용

```html
<section data-module-type="A" data-module-order="1" class="relative w-full overflow-hidden">
  <img src="./images/banner_main.png" alt="{상품명} 상세페이지 배너" class="w-full h-auto object-cover" />
  <div class="max-w-[830px] mx-auto px-5 py-5 bg-white flex items-center justify-between gap-4 flex-wrap">
    <div>
      <h1 class="text-lg font-bold text-gray-900 leading-snug">{상품명}</h1>
      <p class="text-gray-500 text-sm mt-1">{핵심 한줄 카피}</p>
    </div>
    <div class="text-right">
      <p class="text-xs text-gray-400 line-through">{정가}</p>
      <p class="text-red-600 text-2xl font-extrabold">{판매가}</p>
      <span class="inline-block bg-red-500 text-white text-xs font-bold px-2 py-0.5 rounded mt-1">{할인율}% SALE</span>
    </div>
  </div>
</section>
```

## Module C-1: 상품 컷샷 (11번가 전용)
