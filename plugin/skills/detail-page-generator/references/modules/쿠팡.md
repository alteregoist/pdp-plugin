# 쿠팡 모듈 정의

> 공통 템플릿은 `common.md`를 참조한다. 이 파일에 정의된 항목이 common.md보다 우선 적용된다.

## 플랫폼 기본값

- **최대 너비**: 780px → `max-w-[780px]`
- **기본 모듈 순서**: A → B → C-1 → E → C-2 → F
- **필수 모듈**: A, F
- **특이사항**: 이미지 중심, 텍스트 최소화. 모바일 트래픽이 대부분이므로 모바일 최적화 최우선.

---

## Module A: 상단배너 (쿠팡 전용)

**오버라이드 포인트**:
- 배너 이미지 규격: 780 × 440px
- 임팩트 중심 — 큰 이미지, 최소한의 텍스트
- 쿠팡 로켓배송 등 혜택 배지 포함 가능

```html
<section data-module-type="A" data-module-order="1" class="relative w-full overflow-hidden bg-black">
  <img src="./images/banner_main.png" alt="{상품명} 메인 배너" class="w-full h-auto object-cover opacity-95" />
  <div class="absolute bottom-0 left-0 right-0 p-6 bg-gradient-to-t from-black/70 via-black/30 to-transparent">
    <h1 class="text-white text-3xl font-extrabold mb-1 leading-tight drop-shadow">{상품명}</h1>
    <p class="text-white/80 text-base">{핵심 한줄 카피}</p>
  </div>
</section>
```

---

## Module B: 한눈에 보기 (쿠팡 전용)

**오버라이드 포인트**:
- 텍스트보다 아이콘·이미지 비중 높게
- 배지(badge) 스타일로 간결하게 표현
- 쿠팡 오렌지(`#f26522`) 포인트 컬러 활용 가능

```html
<section data-module-type="B" data-module-order="2" class="py-8 px-4 bg-gray-50">
  <div class="max-w-[780px] mx-auto">
    <div class="grid grid-cols-2 md:grid-cols-{N} gap-3">
      <div class="flex flex-col items-center text-center p-5 bg-white rounded-2xl shadow-sm">
        <img src="./images/icon_{N}.png" alt="{특장점}" class="w-10 h-10 mb-3" />
        <h3 class="font-bold text-gray-900 text-sm leading-tight">{특장점 키워드}</h3>
      </div>
    </div>
  </div>
</section>
```

## Module C-1: 상품 컷샷 (쿠팡 전용)

**오버라이드 포인트**:
- 컷샷 이미지 규격: 780 × 780px (1:1) — 정사각형 강조
- 이미지 풀 너비 → 텍스트는 이미지 아래 간결하게
- 텍스트는 핵심만, 2~3줄 이내

```html
<section data-module-type="C-1" data-module-order="3" class="bg-white">
  <div class="max-w-[780px] mx-auto">
    <div class="space-y-1">
      <img src="./images/cutshot_01.png" alt="{설명}" class="w-full h-auto" />
      <div class="px-5 py-4 bg-white">
        <h3 class="text-base font-bold text-gray-900 mb-1">{헤드라인}</h3>
        <p class="text-gray-500 text-sm leading-relaxed">{한두 줄 설명}</p>
      </div>
      <img src="./images/cutshot_02.png" alt="{설명}" class="w-full h-auto" />
      <div class="px-5 py-4 bg-white">
        <h3 class="text-base font-bold text-gray-900 mb-1">{헤드라인}</h3>
        <p class="text-gray-500 text-sm leading-relaxed">{한두 줄 설명}</p>
      </div>
    </div>
  </div>
</section>
```

## Module E: 인포그래픽 (쿠팡 전용)

**오버라이드 포인트**:
- 이미지형 인포그래픽 우선 (텍스트 기반 최소화)
- 비교형 레이아웃 인기 — 경쟁 제품 대비 우위 강조

```html
<section data-module-type="E" data-module-order="4" class="w-full bg-white">
  <img src="./images/infographic.png" alt="{인포그래픽 설명}" class="w-full h-auto" />
</section>
```
