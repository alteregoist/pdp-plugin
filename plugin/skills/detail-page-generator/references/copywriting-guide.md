# 상세페이지 카피라이팅 가이드

상품 상세페이지의 카피는 구매 전환에 직접적인 영향을 미친다.
카테고리와 타깃 고객에 맞는 톤과 메시지를 작성한다.

---

## 카피라이팅 기본 원칙

### 1. 짧고 명확하게
- 헤드라인: 15자 이내
- 서브카피: 30자 이내
- 본문 1문장: 40자 이내
- 불필요한 수식어 제거

### 2. 혜택 중심 (Feature → Benefit)
- BAD: "100% 유기농 면 사용"
- GOOD: "아이 피부에 닿아도 안심, 100% 유기농 면"

### 3. 구체적 수치 활용
- BAD: "오래 가는 배터리"
- GOOD: "한 번 충전으로 72시간"

### 4. 감각적 표현
- 시각: "선명한", "깔끔한", "투명한"
- 촉각: "부드러운", "쫀쫀한", "매끈한"
- 미각: "달콤한", "고소한", "진한"
- 후각: "상쾌한", "은은한", "깊은"

---

## 카테고리별 톤 & 매너

### 식품
- **톤**: 신선, 건강, 자연, 정직
- **키워드**: 엄선된, 갓 수확한, 100%, 무첨가, 자연 그대로
- **헤드라인 패턴**:
  - "{원산지}에서 온 {형용사} {상품명}"
  - "매일 먹는 {상품}, {차별점}으로 다르게"
  - "{수치}시간 내 직송, {상품명}의 신선함"
- **색상 톤**: 그린, 오렌지, 내추럴 브라운

### 가전/전자
- **톤**: 혁신, 편리, 스마트, 프리미엄
- **키워드**: 스마트, 자동, 초절전, 무소음, 올인원
- **헤드라인 패턴**:
  - "{동사}만 하세요, 나머지는 {상품명}이"
  - "일상이 달라지는 {수치}가지 이유"
  - "{기존 불편}은 잊으세요. {상품명}이 있으니까"
- **색상 톤**: 다크 그레이, 블루, 실버

### 생활용품
- **톤**: 따뜻, 실용, 꼼꼼, 안심
- **키워드**: 든든한, 매일 쓰는, 깔끔한, 간편한, 알뜰한
- **헤드라인 패턴**:
  - "매일 쓰는 {상품}, {차별점}이 다릅니다"
  - "{고민}? {상품명} 하나면 끝"
  - "우리 집 {공간}을 바꾸는 {형용사} {상품명}"
- **색상 톤**: 화이트, 민트, 라이트 블루

### 패션/뷰티
- **톤**: 트렌디, 세련, 자신감, 감각
- **키워드**: 데일리, #OOTD, 러블리, 시크한, 글로우
- **헤드라인 패턴**:
  - "오늘의 {아이템}, {브랜드/상품명}"
  - "{계절} 필수템, {상품명}으로 완성"
  - "{형용사} 무드를 담은 {상품명}"
- **색상 톤**: 핑크, 베이지, 블랙, 골드

### 건강/헬스
- **톤**: 신뢰, 전문, 과학적, 케어
- **키워드**: 임상 검증, 특허 성분, 하루 한 포, 꾸준히
- **헤드라인 패턴**:
  - "{타깃}을 위한 {성분} {상품명}"
  - "매일 {수치}{단위}, {효과}의 시작"
  - "{전문가/기관} 인증, {상품명}"
- **색상 톤**: 화이트, 그린, 딥블루

## 모듈별 카피 작성 가이드

### Module A (상단배너)
- 헤드라인: 상품명 + 핵심 차별점 (10~15자)
- 서브카피: 감성적 한줄 (20~30자)
- 예시: "72시간 저온숙성 한우" / "깊은 풍미, 입안 가득 퍼지는 행복"

### Module B (한눈에 보기)
- 특장점 키워드: 2~4자 (예: "무첨가", "저칼로리", "국내산")
- 특장점 설명: 10~20자
- 3~5개로 압축, 중복 내용 제거

### Module C-1 (컷샷)
- 각 컷샷 블록별:
  - 키워드 태그: 영문 또는 국문 2~3자
  - 헤드라인: 10~15자 (혜택 중심)
  - 본문: 2~3줄, 구체적 설명

### Module C-2 (상품 정보)
- 정보 항목은 정확하고 간결하게
- 단위, 수치 명확히 기재
- 필요시 표 형태로 구조화

### Module D (인증 정보)
- 인증 정식 명칭 사용
- 인증 번호 포함 (있는 경우)
- 간결한 부가 설명 (10자 이내)

### Module E (인포그래픽)
- 수치 강조: 폰트 크기 차별화
- 단위 별도 표기
- 비교 데이터는 시각적 대비 명확히

### Module F (표시사항)
- 법적 고지 문구 정확히 기재
- 가독성 위해 항목별 구분
- 주의사항은 별도 강조 박스

## 카피 품질 체크리스트

작성 후 다음을 확인한다:

- [ ] 맞춤법/띄어쓰기 오류 없음
- [ ] 과장 광고 표현 없음 ("최고", "유일한", "완벽한" 등 주의)
- [ ] 타깃 고객에 맞는 톤
- [ ] Feature가 아닌 Benefit 중심
- [ ] 구체적 수치/근거 포함
- [ ] 모듈 간 메시지 일관성
- [ ] 법적 필수 고지 누락 없음
