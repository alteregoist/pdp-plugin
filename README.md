# 🛍️ PDP Plugin

> **Claude Cowork 플러그인** — 상품 이미지와 정보를 입력하면 이커머스 상세페이지를 자동 생성합니다.

## 개요

PDP Plugin은 판매자가 상품 정보만 입력하면 **카피라이팅 → AI 이미지 생성 → HTML 빌드 → 스크린샷 캡처**까지 전 과정을 자동화하는 Claude Cowork 플러그인입니다.

롯데마트몰, 쿠팡, 네이버 스마트스토어, 11번가 4개 플랫폼 규격을 지원하며, 복수 플랫폼 선택 시 병렬 빌드로 한 번에 생성합니다.

```
/상세페이지 유기농 꿀
→ 상품 정보 입력 (카테고리, 플랫폼, 특장점)
→ AI 카피라이팅 자동 생성
→ fal.ai 이미지 생성
→ HTML + TailwindCSS 상세페이지 빌드
→ Playwright 스크린샷 캡처
→ full.png + 모듈별 이미지 출력
```

---

## 주요 기능

| 기능 | 설명 |
|---|---|
| **상세페이지 자동 생성** | 상품 정보 입력만으로 A~F 모듈 조합의 상세페이지 HTML 생성 |
| **AI 카피라이팅** | 카테고리별 톤(식품/가전/생활용품/패션/건강)에 맞는 카피 자동 작성 |
| **AI 이미지 생성** | fal.ai 멀티 모델로 배너·컷샷·아이콘 등 부족한 이미지 자동 생성 |
| **멀티 플랫폼 지원** | 롯데마트몰·쿠팡·네이버 스마트스토어·11번가 플랫폼별 규격 적용 |
| **병렬 빌드** | 복수 플랫폼 선택 시 Task 도구로 동시 빌드, 최대 4배 속도 향상 |
| **모듈 편집** | 생성된 상세페이지의 모듈 단위 텍스트·이미지·순서 편집 |
| **이미지 캡처** | Playwright로 HTML → 전체 페이지 PNG 및 모듈별 PNG 캡처 |

## 설치

Claude Cowork에서 플러그인 폴더를 지정하거나, `plugin/` 디렉토리를 Cowork 플러그인 경로에 복사합니다.

**시스템 요구사항**

- Node.js 18+
- Claude Cowork (로컬 플러그인 지원 버전)
- fal.ai API 키 (이미지 생성 시 필요)

**환경 변수**

```bash
FAL_KEY=your_fal_ai_api_key
```

fal.ai API 키는 [fal.ai 대시보드](https://fal.ai/dashboard/keys)에서 발급받을 수 있습니다.

> npm 패키지(`playwright`, `sharp`)는 첫 실행 시 자동으로 설치됩니다.

## 사용법

### 상세페이지 생성

```
/상세페이지 [상품명]
```

실행하면 대화형으로 아래 정보를 입력합니다.

1. 상품 카테고리 (식품 / 가전 / 생활용품 / 패션 / 건강)
2. 대상 플랫폼 (복수 선택 가능)
3. 핵심 특장점 3~5개

이미지 제공 방법:
- **URL 제공** — `https://...` 이미지 주소를 채팅에 붙여넣기
- **파일 제공** — 작업 폴더에 이미지 파일(`.jpg` / `.png` / `.webp`)을 저장

### 상세페이지 편집

```
/수정하기 [HTML 파일 경로]
```

생성된 상세페이지의 모듈을 편집·추가·삭제·순서 변경합니다.

### 미리보기

```
/미리보기 [HTML 파일 경로]
```

Playwright로 HTML을 렌더링해 전체 페이지 및 모듈별 스크린샷을 캡처합니다.

## 모듈 구성

상세페이지는 아래 8종 모듈을 조합해 구성됩니다.

| 모듈 | 이름 | 설명 | 필수 여부 |
|---|---|---|---|
| **A** | 상단배너 | 메인 비주얼 + 상품명 + 핵심 카피 | 필수 (최상단) |
| **B** | 한눈에 보기 | 핵심 특장점 아이콘 카드 (3~5개) | 선택 |
| **C-1** | 상품 컷샷 | 이미지 + 텍스트 교대 배치 | 선택 |
| **C-2** | 상품 정보 | 텍스트 중심 스펙·성분 정보 | 플랫폼 따라 |
| **D** | 인증 정보 | 인증마크 및 설명 | 식품 필수 |
| **E** | 인포그래픽 | 수치·비교 데이터 시각화 | 선택 |
| **F** | 표시사항 | 법적 고지·주의사항 | 필수 (최하단) |
| **IMG** | 상품 이미지 | 풀너비 이미지 단독 모듈 | 선택 |

## 출력 구조

```
{상품명}/
├── detail_page.html      # 상세페이지 HTML
├── full.png              # 전체 페이지 이미지
├── product_info.yaml     # 상품 정보 (재생성·수정 시 참고)
├── image_log.json        # 이미지 생성 이력
├── images/               # 생성·사용된 이미지
├── originals/            # 사용자 제공 원본 이미지
└── modules/              # 모듈별 캡처 이미지
```

복수 플랫폼 선택 시 플랫폼별 하위 폴더로 분리됩니다.

## 기술 스택

- **HTML + TailwindCSS** (Play CDN) + Paperlogy 웹폰트
- **fal.ai MCP** — AI 이미지 생성·편집
- **Node.js Playwright** — HTML → PNG 캡처
- **Node.js Sharp** — 이미지 리사이즈·포맷 변환
